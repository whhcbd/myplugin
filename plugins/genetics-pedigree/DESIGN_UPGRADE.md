# 家系图插件功能升级设计方案

## 📋 目标
为初高中生物教学设计的遗传家系图插件，严格匹配教材规范、高考要求，形成完整的遗传分析工具链。

---

## 一、核心必选元素（全学段通用）⭐⭐⭐

### 1.1 教材统一的标准符号与配套图例

**当前状态：** ✅ 已实现基础符号（□○■●）

**需要补充的符号：**

#### A. 携带者符号（高考核心）
```
⊙ = 携带者（表型正常的杂合子）
- 常隐：Aa（一个显性一个隐性）
- X隐：X^A X^a（女性携带者）
```

#### B. 特殊状态符号
```
↑ = 先证者（箭头指向，家系中第一个被发现的患者）
⊘ = 已故（符号加斜杠）
◇ = 性别不明（菱形）
□̇ = 收养（虚线边框）
```

#### C. 双胞胎连接
```
倒V连接 = 同卵双胞胎
倒Y连接 = 异卵双胞胎
```

#### D. 编号系统规范
```
世代：罗马数字（Ⅰ、Ⅱ、Ⅲ）
个体：阿拉伯数字（1、2、3）
格式：Ⅱ-3 表示第二代第3个个体
```

**实现优先级：** P0（必须实现）

---

### 1.2 清晰规范的家系结构与世代逻辑

**当前状态：** ✅ 基本实现

**需要强化的规则：**

1. **世代严格纵向排列**：亲代在上，子代在下，不可颠倒
2. **同代水平对齐**：同一世代所有个体Y坐标必须相同
3. **婚配线规则**：横向连接配偶，近亲婚配用双横线
4. **亲子线规则**：从婚配线中点垂直下降，连接子代水平线
5. **子代排序**：从左到右按出生顺序排列

---

### 1.3 明确的性状定义与核心规则标注

**当前状态：** ⚠️ 仅显示性状名称，缺少详细说明

**需要添加"性状说明卡片"：**

```
┌─────────────────────────────────────────┐
│ 性状：白化病（常染色体隐性遗传）         │
│ ■/● = 患者（基因型 aa）                 │
│ ⊙ = 携带者（基因型 Aa，表型正常）        │
│ □/○ = 正常（基因型 AA 或 Aa）           │
│ A = 正常基因（显性），a = 致病基因       │
└─────────────────────────────────────────┘
```

**实现优先级：** P0

---

### 1.4 完整的核心亲缘关系呈现

**当前状态：** ✅ 已实现

**验证要点：**
- 所有世代的亲本、子代完整
- 兄弟姐妹关系清晰
- 先证者的直系血亲无缺失

## 二、进阶拓展元素（高中必加）⭐⭐

### 2.1 高考高频考点专项标注

**优先级：** P0（高考核心）

#### A. 近亲婚配标注
```
显示：双横线（══）
颜色：红色 #ef4444
提示：近亲婚配会显著提高隐性遗传病发病率
```

#### B. 先证者标注
```
符号：↑（箭头指向）
位置：个体符号下方
颜色：蓝色 #3b82f6
标签：P（Proband）
```

#### C. 携带者标注（核心重点）
```
常染色体隐性携带者：
  男性：□ 内加实心点
  女性：○ 内加实心点
  基因型：Aa

X连锁隐性携带者：
  女性：○ 内加实心点
  基因型：X^A X^a
  说明：表型正常但可传递致病基因
```

---

### 2.2 个体基因型与概率标注

**优先级：** P0（高考得分点）

**显示内容：**

1. **确定基因型**：AA, Aa, aa, X^A Y, X^a Y
2. **待定基因型**：A_（下划线表示待定）
3. **概率标注**：
   ```
   示例：Ⅱ-3
         ○
        Aa
       (2/3)
   
   说明：表型正常时，Aa概率 = 2/3
        （排除aa后的条件概率）
   ```

### 2.4 特殊遗传现象的补充标注

**优先级：** P2（拓展内容）

**支持的特殊情况：**

1. **伴Y遗传**：全男性患病，父→子→孙
2. **细胞质遗传**：母病子女全病（线粒体）
3. **显性纯合致死**：AA致死，子代比例2:1
4. **延迟显性**：成年后才发病
5. **新发突变**：父母正常，子女患显性病

### 3.2 高频易错点警示

**优先级：** P1

**错误警示列表：**

```
符号错误：
❌ 禁止颠倒男女符号（方形=男，圆形=女）
❌ 禁止自创符号（必须符合教材规范）

布局错误：
❌ 禁止代际顺序颠倒（亲代必须在上）
❌ 禁止亲子线连错（必须从婚配线中点下降）

计算错误：
❌ 计算Aa概率时忽略"表型正常"前提
❌ 近亲婚配时忽略概率修正
❌ 伴性遗传时忽略性别差异
```

---

### 3.3 与孟德尔方格的联动标注

**优先级：** P2

**功能设计：**
点击婚配对，弹出对应的孟德尔方格图

```
父母：Aa × Aa

    |  A  |  a  |
----|-----|-----|
  A | AA  | Aa  |
----|-----|-----|
  a | Aa  | aa  |

基因型比例：AA:Aa:aa = 1:2:1
表型比例：正常:患病 = 3:1
携带者概率：Aa = 2/3（排除aa）
```

## 六、数据结构设计 📊

```typescript
interface Individual {
  id: string;                       // 唯一标识（如 "I-1"）
  generation: number;               // 世代编号（0开始）
  gender: 'male' | 'female' | 'unknown';
  phenotype: 'normal' | 'affected' | 'carrier' | 'uncertain';
  
  // 基因型信息
  genotype?: string;                // 如 "Aa", "X^A X^a"
  genotypeProb?: number;            // 基因型概率（0-1）
  
  // 关系
  parents?: {
    father?: string;
    mother?: string;
  };
  spouseId?: string;
  
  // 特殊标记
  isProband?: boolean;              // 先证者
  isDeceased?: boolean;             // 已故
  isAdopted?: boolean;              // 收养
  isTwin?: 'identical' | 'fraternal'; // 双胞胎类型
  
  // 婚配类型
  matingType?: 'normal' | 'consanguineous'; // 近亲婚配
  
  // 显示标签
  label?: string;                   // 自定义标签
  age?: string;                     // 年龄
}

interface Generation {
  individuals: Individual[];
}
```
